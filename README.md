# writers-bot-api

Preferred package manager: `bun`.

## Common commands

- Install dependencies: `bun install`
- Start dev server: `bun run dev`
- Build: `bun run build`
- Lint: `bun run lint`
- Test: `bun run test`

## Security auditing

Bun does not currently provide a direct replacement for `npm audit` in this repo workflow.

- Audit dependencies: `bun run audit`
- JSON audit report: `bun run audit:json`
